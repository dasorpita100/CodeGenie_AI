# CodeGenie – AI Code Generation Platform

CodeGenie is an AI-powered web application that converts natural language prompts into complete and executable code. It also provides explanations and relevant keywords to help users understand the generated solutions.


## Features

- AI-based code generation  
- Explanation + keyword extraction  
- Multi-language support (Python, Java, C++, etc.)  
- User authentication system  
- User dashboard (history, usage stats)  
- Admin dashboard (analytics & charts)  
- Feedback system  
- AI fallback (Hugging Face + Ollama)  



## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask (Python)  
- **Database:** SQLite  
- **AI Integration:** Hugging Face API, Ollama  
- **Libraries:** Chart.js, Highlight.js  



## How It Works

1. User selects a programming language  
2. Enters a prompt  
3. Backend processes request  
4. AI generates:
   - Code  
   - Explanation  
   - Keywords  
5. Output is displayed and stored  




##  Purpose

CodeGenie aims to simplify coding and improve understanding by combining AI, learning support, and analytics into a single platform.
